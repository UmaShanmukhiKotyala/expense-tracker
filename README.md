# Expense Tracker (Frontend)

A responsive expense tracker web application built using HTML, CSS, and JavaScript.

## Features
- User login (client-side)
- Monthly budget with auto daily calculation
- Calendar-based expense tracking
- Color-coded spending status
- Prevents future expense entry
- Day-wise and monthly expense views
- Instructions popup for first-time users
- Mobile-friendly design
- Local Storage based data persistence

## Tech Stack
- HTML
- CSS
- JavaScript
- Local Storage

## Note
This version uses client-side storage. Backend integration is planned in a future version.
